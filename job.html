Frontend Developer

Apply Now

Refrence No# : ab246

Salary range : 25k-50k

Interviewer : Adnan Malik

We are seeking a dedicated HTML developer to work with our product team. You will collaborate with designers to create effective and eye-catching website layouts based on Photoshop template files. As an HTML developer, you will be expected to create responsive web designs that match templates for a range of standard screen resolutions. Your duties will include creating HTML and CSS files that can be passed to web developers to create working websites and apps

Roles and Responsibilities:

  • Convert PSD templates into HTML using HTML coding standards.
  • Work on conversion within the given time frames.
  • Create high-quality HTML files based on the latest HTML5 and W3C standards.
  • Use CSS (and CSS preprocessors, such as Less and Sass) to create web applications and app layouts that match designer visions at a range of screen resolutions.
  • Use HTML attributes to create websites that support accessibility options.
  • Use W3C Markup Validation Service and CSS Validation Service to ensure all layouts comply with W3C standards.

Soft Skills:

  • Teamwork 
  • Problem-solving
  • Innovation and best coding practices
  • Interpersonal and communication skills
  • Critical and out-of-the-box thinking
  • Upgrade skills and adapt to changes

preferable:

  • Master's degree
  • 3+ years of experience

Salary range : 25k-90k

Interviewer : Adnan Malik

The ideal candidate will be responsible for end-to-end recruitment efforts, new hire orientation, and onboarding, employee documentation, payroll and benefits, and company policy and procedure adherence.

Roles and Responsibilities:

  • Active participation in full-cycle recruitment for all positions. Hands on expertise in sourcing quality candidates.
  • Assist with payroll and benefits administration.
  • Ensures compliance with company policies and procedures and legal responsibilities.
  • Able to analyze data and present reports whenever required.

  1. Bachelor's or Master's degree in Human Resources.
  2. 2 - 3 years of experience working in Human Resources.

  4. Strong interpersonal and communication skills.
